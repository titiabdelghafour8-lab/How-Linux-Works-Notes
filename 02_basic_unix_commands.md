#Basic unix commands
## I - Introduction
During this chapter we explore the usage of the linux operating system from a human user persepective, how to use its different functionalities, since the commands we are going to explore are inherited from the original unix, and linux is built on top of it the name persisted, those commands are compatible with any other unix like system such as MacOs or BSD.

## II - The shell
the shell is the program that runs our commands, back in unix days they used the bourne shell (/bin/sh) which is the shell developed at Bell Labs, while today they use a more capable and secure version called Bourne-again Shell or _bash_ as a shorthand, today /bin/sh is usually a link to bash rather than Bourne shell. (you can always change your shell using the **chsh** command)
